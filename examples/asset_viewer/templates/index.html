About

This web application demonstrates the capabilities of the ultima_sdk Python library by providing an interactive browser-based interface to explore Ultima Online game assets.

Each asset type has its own dedicated viewer page with search, filtering, and detailed information display capabilities.

Features:
  • Real-time asset loading and rendering
  • Interactive search and filtering
  • High-quality image export capabilities
  • Comprehensive asset information display
  • Responsive web interface
